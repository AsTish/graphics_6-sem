Генератор галактик

Программа генерирует изображения галактик двух типов:

- Миндалевидная
- Спиральная

Реализовано в двух версиях:

1. Графическая версия (GUI) — интерактивная визуализация галактики.
2. Консольная версия (CLI) — интерактивное меню для задания параметров и сохранения изображения.

---

1. Графическая версия (GUI)

Файл запуска: galaxy_window.exe (в папке dist)

Возможности:

- Выбор типа галактики: Миндалевидная или Спиральная
- Миндалевидная галактика:
  - Регулировка угла наклона с шагом ±5° с помощью кнопок
  - Радиус: задаётся пользователем
- Спиральная галактика:
  - Выбор количества рукавов (1–10)
  - Радиус фиксированный: 400
  - Длина рукавов: 300
  - Ширина рукавов: 100
- Выбор цвета центра и края галактики (через диалоговое окно цвета)
- Строка состояния показывает текущие параметры: тип галактики, радиус, цвет центра, цвет края, количество рукавов (для спиральной)
- Фон изображения: фиксированный черный (1000x1000 пикселей)

---

2. Консольная версия (CLI)

Файл запуска: galaxy_cli.exe (в папке dist)

Возможности:

1. Выбор типа галактики: Миндалевидная или Спиральная
2. Настройка параметров:
   - Миндалевидная: угол наклона
   - Спиральная: количество рукавов
3. Выбор фона:
   - Черный
   - Белый
   - Своя картинка
4. Выбор цвета центра и края галактики из 10 предустановленных цветов:
   - White, Blue, Red, Yellow, Purple, Cyan, Orange, Green, Pink, Light Blue
5. Ввод имени файла для сохранения. Если расширение не указано, используется .jpg

Параметры галактик:

- Миндалевидная: a=280, b=100
- Спиральная: radius=400, arm_length=300, arm_width=100
- Размер создаваемого изображения: 800x800 пикселей

---

Структура проекта

galaxy.py                – базовый класс GalaxyBase
lenticular_galaxy.py     – миндалевидная галактика
spiral_galaxy.py         – спиральная галактика
galaxy_window.py         – графический интерфейс
galaxy_cli.py            – консольное приложение
Pipfile / Pipfile.lock   – зависимости проекта
dist/                    – папка с готовыми exe
